Mac 解决`.bash_profile`配置项不生效问题

2019-11-27 / 1 min read

前言

重启电脑，打开iTerm2敲命令的时候，突然发现配置项不生效了，要source ~/.bash_profile 一下才行，好像每次重启了都要重载一下命令。这就令人很头疼啦！网上查了一下，原来是我的iTerm2是加载~/.zshrc文件，然而这个文件里面并没有定义到环境变量导致的。

解决

其实解决办法很简单，只需在~/.zshrc文件下面添加重载命令即可

# 打开 .zshrc 文件
vim ~./zshrc

# 在最后添加命令
source ~/.bash_profile

# 保存退出
:wq

# 重载 `.zshrc` 文件
source ~./zshrc
