URL参数截取工具函数

· 2019-05-04 · # URL # JavaScript # 前端

在地址栏截取相应的参数这种场景还是比较常见的，在这里和大家分享一下这个工具函数

// name 的值就是你想要获取的key值的名称
function getUrlKey(name) {   
            return decodeURIComponent((new RegExp('[?|&]' + name + '=' + '([^&;]+?)(&|#|;|$)').exec(location.href) || [, ""])[1].replace(/\+/g, '%20')) || null
}
